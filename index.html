What is OCKB?

The Open Class Knowledge Base (OCKB) is an application designed to pool the knowledge for a specific class into one accessible place. Users are able to post questions as well as answers to topics being covered in the class in a style similar to bulletin boards or forums. To access the application, you would need a valid University of Hawaii account. If you don't have one, you are still able to explore the site, but without the ability to post or see the identities of the posters.
